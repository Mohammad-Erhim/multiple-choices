## About
multiple-choices : it is a quiz app built with React & NodeJs .

## Scenarios 

- login
- sign up
- log out 
- add question with the right answer
- go into the exam 
- set question number
 
> PS: this app is not stable and it's only for preview
 
## Installation and Run

- back-end
```sh
npm i
npm start
``` 
- front-end 
```sh
cd client
npm i
npm start
``` 

## Test

```sh
cd client
npm test
```
